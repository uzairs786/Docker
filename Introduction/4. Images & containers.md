## ğŸ§° Dockerfile (How Images Are Created)

A **Dockerfile** is a text file containing instructions used to build an image.

It typically includes:

* The base image to use (`FROM`)
* Files to copy into the container (`COPY`)
* Commands to run inside the image (`RUN npm install`, etc.)
* Default commands for the container (`CMD`, `ENTRYPOINT`)

Docker reads the Dockerfile **step-by-step** to assemble an image.
